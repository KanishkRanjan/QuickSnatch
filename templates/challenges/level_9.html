Level 9: System Services

9/10

System services and scheduled tasks are critical components of Linux systems.

Your mission:

  1. Check status of flag-service using systemctl
  2. View scheduled tasks with crontab -l
  3. Monitor service logs with journalctl
  4. Extract the flag from service logs
{% include 'terminal.html' %}
Available Commands:
  • systemctl status service - Check service status
  • crontab -l - List scheduled tasks
  • journalctl -u service - View service logs
  • tail -f /var/log/syslog - Monitor system logs
Tips:
  • Services often write to system logs
  • Check cron jobs for scheduled tasks
  • Use journalctl to view service history
