Welcome to Level 1

Basic Terminal Navigation

As a security analyst, you've gained initial access to a suspicious system. Your first task is to gather basic intelligence using terminal commands.
Mission Objectives
1
Establish Location

Use pwd to determine your current position in the system

2
Reconnaissance

Execute ls -la to identify all files, including hidden ones

3
File Analysis

Look for suspicious or hidden files that might contain sensitive information

4
Flag Extraction

Use appropriate commands to read and extract the hidden flag

Intelligence Report
  • Target system appears to be a Linux-based environment
  • Previous reconnaissance suggests the presence of hidden files
  • The flag may be hidden in plain sight or require basic file inspection
Field Notes
  • Hidden files in Linux start with a dot (.)
  • The ls -la command shows all files with detailed information
  • Use cat to read file contents
  • Pay attention to file permissions and ownership
