Level 4: Process Investigation

4/10

Process management is a crucial skill for system administrators.

Your mission:

  1. List all running processes using ps aux
  2. Find suspicious processes
  3. Investigate process details in /proc directory
  4. Extract the flag from process information
{% include 'terminal.html' %}
Available Commands:
  • ps aux - List all processes
  • ps -p PID - Show specific process info
  • cat /proc/PID/cmdline - View process command line
  • strings /proc/PID/environ - View process environment
Tips:
  • Look for processes with suspicious names or high resource usage
  • The /proc directory contains detailed process information
  • Process environment variables may contain sensitive data
