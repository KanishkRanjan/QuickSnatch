Level 3: Log Analysis

3/10

System administrators often need to analyze log files to find important information.

Your mission:

  1. Navigate to the logs directory
  2. Examine different types of log files
  3. Use grep to search for ERROR messages
  4. Find and extract the flag from the logs
{% include 'terminal.html' %}
Available Commands:
  • ls - List files and directories
  • cd directory - Change directory
  • cat file - Display file contents
  • grep pattern file - Search for pattern in file
  • zcat file.gz - View compressed files
Tips:
  • Use grep to search for specific patterns in files
  • Some log files might be compressed (.gz)
  • Look for ERROR messages in the logs
