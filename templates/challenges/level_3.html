Welcome to level 3! Time to master log analysis using grep.

Your mission:

  1. Navigate to the logs directory
  2. Analyze various log files (error.log, access.log, system.log)
  3. Use grep to search for the flag pattern
  4. Find and decode the hidden flag
Grep Tips:
  • Use grep pattern file to search in a single file
  • Use grep -r pattern directory to search recursively
  • Use grep -i pattern for case-insensitive search
  • Use grep -A 2 pattern to show 2 lines after match
  • Use grep -B 2 pattern to show 2 lines before match
