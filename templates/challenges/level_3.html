Finding Patterns in Logs

Your task is to search through various log files to find a hidden flag. You'll need to use the grep command effectively.

Your Tasks:
  • Navigate through the logs directory
  • Examine different types of log files (error.log, access.log, debug.log)
  • Use grep to search for the flag pattern
  • Try recursive search with grep -r

Hint: The flag might appear in multiple log files. Use grep -r to search recursively through all files in a directory.
