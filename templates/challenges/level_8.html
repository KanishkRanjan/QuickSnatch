Available Commands:
  • top - Monitor processes in real-time
  • ps aux | grep pattern - Search processes
  • cat /proc/PID/status - View process status
  • strings /proc/PID/environ - View process environment
  • cat /proc/PID/cmdline - View process command line
Tips:
  • Look for processes with unusual CPU usage
  • Check process environment variables
  • Examine command line arguments
