File Permissions

In this challenge, you'll need to understand and manipulate Linux file permissions to access a protected file.

Your Tasks:
  • Use ls -l to view file permissions
  • Read and understand the instructions in instructions.txt
  • Study the permissions information in permissions_info.txt
  • Modify permissions of secret.txt to read its contents

Hint: The chmod command changes file permissions. Remember the numeric values: read (4), write (2), execute (1).
