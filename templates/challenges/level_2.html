Available Commands:
  • ls -l - List files with permissions
  • cat filename - Display contents of a file
  • chmod permissions filename - Change file permissions
Permission Numbers:
  • 4 = read (r)
  • 2 = write (w)
  • 1 = execute (x)
  • Example: chmod 644 file (rw-r--r--)
