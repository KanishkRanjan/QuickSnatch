Welcome to level 2! Let's learn about file permissions in Linux.

Your mission:

  1. Check file permissions using ls -l
  2. Find a file with restricted permissions
  3. Use chmod to modify permissions
  4. Read the flag from the file
Understanding Permissions:

Linux permissions are represented as: rwx rwx rwx (user, group, others)

  • r - Read permission (4)
  • w - Write permission (2)
  • x - Execute permission (1)
{% endblock %} {% block available_commands %}
  • ls -l - List files with permissions
  • chmod - Change file permissions
  • chmod +r file - Add read permission
  • chmod +w file - Add write permission
  • chmod +x file - Add execute permission
  • cat file - Read file contents
